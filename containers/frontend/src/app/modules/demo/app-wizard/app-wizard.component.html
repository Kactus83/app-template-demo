Bêta

Assistant d’installation

Simplifiez l’installation et la configuration de votre environnement de développement : Node.js, npm, GitHub CLI, Docker Desktop… Le tout via une interface graphique intégrée.

Disponible pour l’instant uniquement sous Windows 10 et versions ultérieures embarquant WebView (basé sur Tauri). Cet assistant est en phase bêta : il peut rencontrer des difficultés de détection selon vos permissions ou votre configuration. Nous travaillons activement à son amélioration et à son portage sur macOS/Linux.
